##poj3274

这题不太会~看了discuss还有别人的写了个…
原理倒是懂了….
题意也看了好久….
题意是说农场主有N头牛，每头牛都有它的特长….将一个数用K位二进制表示，当某位数为一，则表示该牛有这个特长，例：13的二进制为1101则该牛有特长1,3,4,从右向左……
然后给出一个牛的序列，求一个最大区间—-能够使牛的各个特点的和都为一样，求这个最大区间长度…

首先：
7 37672142
转化二进制：
1 1 1
1 1 0
1 1 1
0 1 0
0 0 1
1 0 0
0 1 0
进行累加：
1 1 1
2 2 1
3 3 2
3 4 2
3 4 3
4 4 3
4 5 3
显然第二项和第六项是最大区间因为
（4,4,3）-（2,2,1）= （2,2,2）
在这个区间里各项和都相等…..即平均， balanced

再进行推导：
因为（a,b,c） - (e,f,g) = x * (1,1,1)

即 （a,b,c）- (a-x,b-x,c-x) = 0

将每行的每个数减上最右边的数。。。
（a-c,b-c,0）- (a-c,b-c,0)
这样我们就只要比较两行数是否相等就行了….
然后得出最大距离

0 0 0
1 1 0
1 1 0
1 2 0
0 1 0
1 1 0
1 2 0
最后还要注意一个边界条件…
即全为1的情况….
2 2
1
2
答案是 2
